Welcome to FIM Agent, an AI-powered framework for building agents that dynamically plan and execute complex tasks across your enterprise systems.

## What is FIM Agent?

FIM Agent is a provider-agnostic Python framework for building AI agents that work with your existing systems. Unlike workflow builders that ask you to replicate logic, FIM Agent bridges your systems proactively вҖ” reading databases, calling APIs, pushing notifications вҖ” all through a unified AI interface.

The core insight: **three delivery modes, one agent core**.

## Three Delivery Modes

| Mode | What it is | Delivery | Use Case |
|------|-----------|----------|----------|
| **Standalone** | General-purpose AI assistant вҖ” search, code, knowledge base | Portal | Chat, code execution, knowledge base Q&A |
| **Copilot** | AI embedded in a host system вҖ” works alongside users in their existing UI | iframe / widget / embed | "Finance Copilot" in your ERP web UI |
| **Hub** | Central cross-system orchestration вҖ” all your systems connected | Portal / API | Agent queries ERP, checks OA, notifies via Lark |
